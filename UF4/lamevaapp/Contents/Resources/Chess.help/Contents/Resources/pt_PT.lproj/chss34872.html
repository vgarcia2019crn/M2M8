Ajuda do Xadrez

Para desafiar o seu Mac para uma partida de xadrez, seleccione Jogo > Novo.

Abrir Xadrez

  • Obter dicas: seleccione Jogadas > “Mostrar dica”. Uma seta indica para onde deve mover a peça. No nível de dificuldade “Mais depressa”, as dicas não estão disponíveis.

  • Anular uma jogada: seleccione Jogadas > “Anular jogada” em relação a cada jogada que pretenda desfazer.

  • Ver jogadas: seleccione Jogadas > “Mostrar última jogada”; uma seta aponta da localização original da peça para o quadrado para onde foi movida. Para ver todas as jogadas de um jogo, seleccione Jogadas > “Registo do jogo”.

  • Alterar o aspecto de um jogo: seleccione Xadrez > Preferências e, depois, seleccione um estilo para o tabuleiro e para as peças.

  • Definir o nível de dificuldade: seleccione Xadrez > Preferências e, depois, arraste o nivelador para “Mais rápido” ou Melhor para diminuir ou aumentar a dificuldade.

  • Ouvir as jogadas enunciadas: seleccione Xadrez > Preferências, assinale as opções relativas às jogadas que pretende ouvir enunciadas e, depois, seleccione as vozes.

  • Alterar o ângulo de apresentação do tabuleiro: clique num canto do tabuleiro e, com o rato ou trackpad, arraste para o ângulo desejado.

  • Usar comandos de voz: para poder usar os comandos de voz, primeiro tem de activar o Ditado Melhorado. Em seguida, seleccione Xadrez > Preferências e seleccione “Permitir que o jogador enuncie as jogadas”. Quando for apresentada a janela Ditado, prima a tecla de atalho (a tecla que escolheu quando activou o Ditado Melhorado) e enuncie um comando; procure não fazer pausas durante a enunciação.

É possível usar comandos enunciados para as seguintes acções:

Acção

Exemplo de comando enunciado

Deslocar uma peça

“Peão e2 para e4”

Capturar uma peça

“Peão e5 captura f6”

Mover uma peça

“Mover bispo para g4” (só em jogos crazyhouse)

Promover um peão

“Peão e7 para e8 promovido a dama”

Roque

“Roque do lado do rei” ou “Roque do lado da dama”

Anular uma jogada

“Anular uma jogada”
